Calendario de Evaluaciones es un proyecto API REST desarrollado por los alumnos Felipe Mancilla, Fabián Da Silva e Ignacio Oneto para el ramo "Desarrollo web orientado a APIs y microservicios", y consiste en una aplicación web que gestiona eventos a través de un calendario para planificar y organizar las evaluaciones que pueda tener una persona con el fin de llevar un correcto orden de estas.
